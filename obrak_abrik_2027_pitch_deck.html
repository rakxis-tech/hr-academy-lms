API USAGE · DEEP DIVE

DeepSeek vs Claude — Kalkulasi Nyata

DeepSeek V4-Flash (Primary)

Input (cache miss)$0.14 / 1M token
Input (cache hit)$0.0028 / 1M token
Output$0.28 / 1M token
Context window64K token

Gunakan untuk: chatbot Q&A materi, ringkasan modul, adaptive quiz, WhatsApp funnel

Claude Haiku 4.5 (Fallback)

Input$1.00 / 1M token
Output$5.00 / 1M token
KelebihanBahasa Indonesia lebih natural

Gunakan untuk: evaluasi esai, feedback kualitatif kompleks, konten kurasi premium

Estimasi Token Usage per Use Case

Chatbot tanya materi (1 query)~500–1K token
Generate ringkasan modul (1 modul)~2–4K token
Adaptive quiz 5 soal~1.5–3K token
WhatsApp lead nurturing (1 flow)~800–1.5K token
Debugging & dev iteration (per sesi)~5–20K token

💡 Strategi Caching — Kunci Efisiensi

System prompt + kurikulum context di-cache prefix. Pertanyaan berulang hit cache = $0.0028 vs $0.14. Hemat 98% untuk query berulang. Raka set ini dari day-1.
